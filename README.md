# Text-Size-Tool
Petit outil interactif permettant d’augmenter ou de réduire la taille d’un texte grâce à des boutons de contrôle. Ce projet m’a permis de pratiquer la manipulation du DOM et la gestion d’événements en JavaScript pour modifier dynamiquement l’affichage d’une page web.  ⚙️ Technologies : HTML • CSS • JavaScript
